# Simple-Lstore

A lightweight educational implementation of an L-Store-style database engine in Python.

## Overview

This project implements core database components for learning and experimentation, including table storage, record operations, and query execution.

## Features

- Table creation and management
- Record insertion and update
- Point/range query support (based on your implementation)
- Delete operations
- Modular design for storage and query layers
- Milestone test script support (`m1_tester.py`)
